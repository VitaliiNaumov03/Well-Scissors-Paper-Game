# Well-Scissors-Paper

It's like a classic rock-paper-scissors, but with well instead of rock.

## Rules

- **Well** beats **Scissors** *(they drown in it)*
- **Scissors** beat **Paper** *(they cut it)*
- **Paper** beats **Well** *(it covers it)*
- If signs are the same, it's a draw
